= 第七部分 模板和泛型编程

== 41. 理解隐式接口和编译期多态

模板化的类和函数，内部的调用取决于类型，没有明确的类型可以看接口有哪些，叫做隐式接口

. class：显式接口，运行时多态
. template: 隐式接口，编译时多态

== 42. 理解typename的两种含义

除了表示类型名，还可用来用来声明嵌套类型，辅助解析器和编译器，让其知道是一种类型，而不是其他的东西

== 43. 知道在模板化的基类中，如何访问名称

模板继承的派生类中，若要访问基类函数，要写this->,或用using导入基类函数

== 44. 与参数无关的代码，应放到模板外

减低模板膨胀的问题，因为很多模板生成的代码，某些函数实际上是重复的
